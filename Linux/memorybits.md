# Memory Bits

A byte is 8 bits, or 2 nibbles (2^3)

*   1kb => $2^10$ or 1'024 bytes
*   1mb => $2^20$ or 1'048'576 bytes
*   1gb => $2^30$ or 1'073'741'824 bytes
*   1tb => $2^40$ or 1'099'511'627'776 bytes
